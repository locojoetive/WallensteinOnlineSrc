Hallo meine lieben Freunde,
dieses SPiel ist fast fertig. Um dieses Spiel fast zu spielen entpacken Sie die zip-Datei und öffnen Sie die Server.jar und Client.jar in dieser
Reihenfolge über die Konsole, mit Hilfe der Befehle `java -jar Server.jar` und `java -jar Client.jar` vergessen Sie nicht das Verzeichnis
zu öffnen in der sich die beiden Dateien befinden. (cd "name des ordners")
Bei der Ausführung der Server datei wird eine ip angezeigt, die Sie euch aufschreiben sollten. Die Server Datei muss jedoch nur einmal ausgeführt
werden. Bei der Ausführung der der Client Datei wird zunächst die ip abgefragt, hier sollte die vorher aufgeschriebene ip eingesetzt werden.
Daraufhin können Sie einen Account erstellen und sich damit einloggen.
Sie können dieses Spiel zu dritt, mittels des Multiplayer buttons, oder alleine, mittels des Singleplayer buttons, (fast) spielen.
Ich hoffe trotzdem Sie haben Spaß dabei und versuchen Sie sich mit einem falschen Passwort anzumelden.
